Hello this page is an attempt of creating an interactive music page


  • Home takes you to the 'Home Page' --> You are on it!!!
  • This web application allows you to register and login
  • Upon registering and logging-in, you can unlock extra features, such as reviewing a track
  • Browse Tracks allows you to search through the database by tracks(alphabetically sorted by track title)
  • Browse Artists allows you to search through the database by artists(alphabetically sorted by artist name)
  • Browse Albums allows you to search through the database by album(sorted by the album's unique id)
  • Browse Genres allows you to search through the database by genre(alphabetically sorted by genre name)
